# Forme for Theia

**Input :** Dataset created by Theia and Hyperion
**Output :** Reshaped dataset

### Configuration

*(optional)*

Open `config.yml` to edit the configuration

```yaml
input:
  folder: input

output:
  size: 32
  folder: output
```

### Install

```shell
pipenv shell
pipenv install
```

### Run

```shell
pipenv shell
python forme.py
```
